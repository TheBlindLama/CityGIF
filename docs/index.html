Privacy Policy - City+

Effective date: February 11, 2026

This extension is designed to work on HabboCity Nitro pages (habbocity.me and habbocity.fr) and provides custom emoji/GIF chat features.

Data We Process

  • Account identifier data you enter in the extension (such as username).
  • PIN data you enter for contributor/admin verification.
  • Emoji/GIF commands and related moderation actions (for example add/remove custom GIFs).
  • Local extension data stored with Chrome storage APIs (session/preferences needed for features).

How We Use Data

  • To authenticate users for protected actions.
  • To sync shared emoji/GIF content through Supabase.
  • To make chat features work (rendering custom emoji/GIFs and related UI actions).

Where Data Is Sent

  • Supabase services configured by the extension project owner.
  • HabboCity pages where the extension is active.

What We Do Not Do

  • We do not sell personal data.
  • We do not run third-party advertising trackers in the extension.
  • We do not request permissions unrelated to extension functionality.

Data Retention

  • Data retention is controlled by the project database configuration (Supabase tables, logs, and policies).
  • Local browser storage remains until you clear extension data or uninstall the extension.

Your Choices

  • You can stop all data processing by disabling or uninstalling the extension.
  • You can clear local extension data from your browser settings.

Contact

Updates

This policy may be updated when features or data flows change. The latest version should be available at the extension listing/support page.
